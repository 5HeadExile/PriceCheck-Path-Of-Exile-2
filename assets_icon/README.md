# Иконка приложения — PriceCheck PoE2

Готовый набор иконок из мастер-изображения (бронзовый лик Divine Orb под лупой
на тёмном медальоне). Фон вырезан в настоящую прозрачность (альфа-канал).

## Содержимое
- **`PriceCheck.ico`** — мультиразмерный значок Windows (внутри 16/32/48/64/128/256px,
  PNG-сжатые). Это основной файл для приложения — Windows сам берёт нужный размер.
- `PriceCheck_master.png` — мастер 1024×1024 с прозрачностью (для App Store/витрин/крупного).
- `icon_512.png … icon_16.png` — отдельные PNG нужных размеров (таскбар, трей, About, сплэш).

## Подключение в WinForms (.NET)

### Значок главного окна и в таскбаре
В дизайнере формы: свойство `Icon` → выбрать `PriceCheck.ico`.
Или кодом:
```csharp
this.Icon = new Icon("PriceCheck.ico");
// либо из встроенных ресурсов:
this.Icon = Properties.Resources.PriceCheck;
```

### Иконка в системном трее (NotifyIcon)
```csharp
notifyIcon1.Icon = new Icon("PriceCheck.ico", new Size(16, 16)); // возьмёт 16px-кадр из .ico
notifyIcon1.Visible = true;
```

### Иконка .exe (отображается в Проводнике)
Назначается в свойствах проекта: **Project → Properties → Application → Resources →
Icon and manifest → Icon** → `PriceCheck.ico`. Пересобрать — Проводник покажет значок у .exe.

### Встроить как ресурс (рекомендуется, чтобы не зависеть от файла на диске)
1. Добавь `PriceCheck.ico` в проект, в свойствах файла **Build Action = Embedded Resource**
   (или через Resources.resx как `Properties.Resources.PriceCheck`).
2. Используй `Properties.Resources.PriceCheck`.

### Про DPI
`.ico` содержит все размеры, поэтому на 125/150% Windows подберёт чёткий кадр сам —
дополнительного масштабирования не нужно. Для крупных мест (сплэш, About-диалог)
бери `icon_512.png` или `PriceCheck_master.png`, а не растягивай 48px.

## Примечание
Фон удалён флуд-заливкой от краёв; мягкая тень-ореол вокруг плашки оставлена
намеренно (читается как естественная тень иконки). Если нужна абсолютно «обрезанная»
версия без тени — скажи, отдам вариант с жёстким контуром.
